## Installation

1. Open the **Extensions** sidebar panel in VS Code. `View â†’ Extensions`

2. Search for `Luvia`

3. Click `Install`

4. When prompted, select `Luvia Theme` as the color theme

> In case of not being prompted to select a Color Theme upon installing, go to the menu bar and select: `Code (File, on Windows) > Preferences > Color Theme > Luvia Theme`. Alternatively, you can use the shortcut `âŒ˜/Ctrl + K > âŒ˜/Ctrl + T` and select `Luvia Theme`.
